2.1 Estructura

Son etiquetas que ayudan a estructurar el diseño de la web. Tienen reglas acerca de su uso adecuado y de interacción con otras etiquetas estructurales.
La mayoria soporta atributos globales como style, en caso contrario se indicará.

body

La etiqueta body posee el contenido del documento html, debe haber un solo body.

main

Posee el contenido principal del documento, debe haber un solo main que desciende de body y no puede descender de etiquetas article, aside, footer, header o nav.
No debe contener nada que se repita en otros sitios del proyecto web como información de contacto, copyright, logos, barras de busqueda, etc.

header

Usualmente posee contenido introductorio o links de navegación, etiquetas heading y/o logo. No puede ser utilizado dentro de una etiqueta footer, adress o header.

nav

Contiene links de navegación como menus, tablas de contenido o índices. El índice se encuentra aquí dentro.

section

Una sección del documento. Cada link dentro de x.x en el índice corresponde a un section.

article

Una parte del documento que se entiende en si mismo. La parte de ejercicios esta en un article, podria ser su propio documento html.

aside

Contiene contenido tangencial al documento, suele aparecer al costado del navegador. El índice esta contenido en un aside en esta página.

footer

Pié de página de un documento o sección.

2.2. HTML, head y meta etiquetas

La etiqueta HTML debe englobar todo el documento. Puede ser utilizada para especificar el idioma del documento: <html lang="es">


El elemento head contiene metadata; título, caracteres, styles, etc


<meta charset="UTF-8"> Hace el documento compatible con caraceteres "UTF-8" (Unicode). Otros set de caracteres no son recomendados.


<name="viewport" content="width=device-width, initial-scale=1"> Ajusta el zoom de la web al abrirla. Necesario para compatibilidad con mobiles


El atributo name y content pueden ser usados para usados para guardar keywords, autor, descripción, etc.

2.4. Links

Tipos de links

La etiqueta a define un hyperlink, que es usado como método de enlace.

Atributos

  • href="url"
  • download="filename"
  • target=... global,parent,top,self. Indica donde abrir el enlace.

    • Links y elementos visuales

    Se puede utilizar una imagen como un link.

    Se puede utilizar un botón como un link.

    Código <a href="ruta" target="_blank"> Imagen/botón</a>

    Id

    El id es un atributo global, entre otras cosas, se puede utilizar como ancla para un link.

    2.5. Elementos visuales

    Imágenes

    Etiqueta img con src como source. Se debe usar el atributo alt por accesibilidad junto a una descripcion. Atributos height y width no de css se deben usar. Loading=(eager/lazy) da condiciones de cuando se debe cargar.

    Figuras

    La etiqueta Figure puede usarse junto con el atributo Figcaption para agregar una descripción a una figura.

    Botones

    Etiqueta button. Se puede usar en formularios con type, en vez de input. Se le puede poner texto dentro.

    Iframes

    Es usado para poner otra pagina. Tiene src y title (Buena practica). Un iframe puede ser usado como el target de un link.
